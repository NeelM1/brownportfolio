# Assets Directory

This directory contains all game assets. For the production version, replace these placeholders with actual assets.

## Required Assets

### Sprites (/sprites/)
- `robotics.png` - Robotics icon (32x32 or 64x64)
- `coding.png` - Coding/programming icon (32x32 or 64x64)
- `tinkering.png` - Tinkering/tools icon (32x32 or 64x64)
- `glitch.png` - Glitch effect sprite (32x32)
- `window_chrome.png` - Window chrome background (260x180)
- `loading_bar.png` - Loading bar background (200x20)
- `click_to_play.png` - Click to play button texture
- `title_bg.png` - Title screen background
- `title_logo.png` - Title logo
- `bedroom_bg.png` - Bedroom background
- `bed.png` - Bed sprite
- `desk.png` - Desk sprite
- `pc.png` - Computer sprite
- `poster.png` - Wall poster sprite
- `door.png` - Door sprite (2 frames: closed, open)
- `snake_bg.png` - Snake world background
- `snake.png` - Snake sprite sheet (16x16 frames)
- `apple.png` - Apple sprite sheet (12x12 frames)
- `scratch_bg.png` - Scratch world background
- `roblox_bg.png` - Roblox/voxel world background
- `npc1.png`, `npc2.png`, `npc3.png` - NPC sprites (32x32 frames)
- `friend1.png`, `friend2.png`, `friend3.png`, `friend4.png` - Friend sprites (32x32 frames)
- `campus_bg.png` - Campus background
- `tree.png` - Tree sprite
- `building.png` - Building sprite
- `path.png` - Path sprite
- `player.png` - Player character sprite sheet (32x32 frames, down/left/right/up animations)
- `cursor.png` - Mouse cursor sprite

## Asset Sources

Recommended CC0 asset sources:
- Kenney.nl: https://kenney.nl/assets

- Incompetech (music): https://incompetech.com/music/
